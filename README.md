## 🚀 Kurulum Adımları

Aracı kullanmak istediğiniz işletim sistemine uygun adımları takip edin.

### 1. Termux Kurulumu (Android)

```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/qyroxr/Rom-Bomb.git
cd Rom-Bomb
pip3 install -r requirements.txt
python3 rom.py
```

---

### 2. Kali Linux Kurulumu

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install git python3 python3-pip -y
git clone https://github.com/qyroxr/Rom-Bomb.git
cd Rom-Bomb
pip3 install -r requirements.txt
python3 rom.py
```

---

### 3. Windows Kurulumu

```bash
git clone https://github.com/qyroxr/Rom-Bomb.git
cd Rom-Bomb
pip install -r requirements.txt
python rom.py
```

---

### 4. iSH Shell Kurulumu (iOS)

```bash
apk update
apk add git python3 py3-pip build-base python3-dev
git clone https://github.com/qyroxr/Rom-Bomb.git
cd Rom-Bomb
pip3 install -r requirements.txt
python3 rom.py
```
